Mi Portafolio

Meet&Coffee

El desafío consta de aplicar los conceptos aprendidos de HTML y CSS, construyendo dos páginas del proyecto Meet&Coffe. Se utilizan dos tipografías con sus respectivos pesos, colores e íconos de Font Awesome, todo esto siguiendo un layout presentado en el desafío.

Iguana Page

El desafío se trata de construir una página desde cero, practicando los conceptos de HTML y CSS. Iguana Page cuenta con distintas fotografías que deben ser colocadas en el DOM, con sus tamaños correspondientes, para que la página quede lo más similar a la maqueta presentada.

Suricata

Este desafío reune los conceptos estudiados de HTML, CSS y Bootstrap, complementandolos y poniendolos en práctica en este desafío. La idea es construir una página con layout responsivo, considerando la filosofía Mobile First, utilizando distintas clases utilitarias de Bootstrap.

Cuppon

Aplicando todos los conceptos y herramientas aprendidas se construye la página de "Cuppon", guiándonos solamente por el mockup de la vista mobile. El concepto más importante en este caso es el de Mobile First, donde lo esencial es apuntar a diseñar prioritariamente la visualización y estructura del sitio en dispositivos móviles.

Ricomida

Este desafío permitió practicar las habilidades adquiridas en relación con HTML, CSS, Bootstrap y jQuery. Consiste en la construcción de una página web relacionada con la preparación de comidas, la cual se debe adapatar a cualquier dispositivo. Utilizamos componentes de Bootstrap que requieren incorporación de JQuery, además de las interacciones que se activaran con eventos provocados por el usuario.

Mi Portafolio

Esta última prueba reune todos los conceptos aprendidos en esta étapa de Fundamentos de Desarrollo Web. Se construye un landing page, este será un portafolio web que integre todas los desafíos realizados anteriormente. Debe ser responsivo, tener una grilla, componentes y clases utilitarias de Bootstrap, con y sin Javascript.
